# FestivalMusica
Este es el primer proyecto de mi portafolio en el cual integro diferentes tecnologías además de HTML y CSS, en el proyecto utilizo JavaScript para la galería de imágenes y el menú de navegación en móviles, tambien utilizo SASS para los estilos y GULP para automatizacion de tareas como generar imagenes en diferentes formatos como webp y avif, comprimir y crear el sourcemap de los archivos de css y javascript. Tambien es el primer proyecto en el que incluyo video.
